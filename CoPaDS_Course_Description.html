Catalog Description

This course is an introduction to the organization and programming of systems comprising multiple computers. Topics include the organization of multicore computers, parallel computer clusters, computing grids, client-server systems, and peer-to-peer systems; computer networks and network protocols; network security; multithreaded programming; and network programming. Programming projects will be required.

Prerequisites

Prerequisite knowledge:

  • Able to read and write computer programs in Java
  • Able to read computer programs in C

Prerequisite courses (or permission of instructor):

  • CSCI 243—The Mechanics of Programming

Course Outcomes

  • The student will explain the concepts of processes, threads, and scheduling. 
    Assessment method: Exams, homework.

  • The student will develop multithreaded programs. 
    Assessment method: Programming projects.

  • The student will explain the concepts of computer networking, the layered network architecture, network security, and network communication with connections and datagrams. 
    Assessment method: Exams, homework.

  • The student will develop network application programs. 
    Assessment method: Programming projects.

  • The student will explain the concepts of distributed system architectures and middleware. 
    Assessment method: Exams, homework.

  • The student will explain the concepts of parallel computer architectures and middleware. 
    Assessment method: Exams, homework.

CS Program Outcomes

Students graduating from our undergraduate program are able to:

  • (CS Undergraduate Program Outcome 2) Demonstrate fluency in high-level programming languages, environments, and tools for computing.

  • (CS Undergraduate Program Outcome 3) Demonstrate knowledge of the principles of computer organization, operating systems, and networks.

  • (CS Undergraduate Program Outcome 7) Comprehend and analyze both legal and ethical issues involving the use of computing in society.
